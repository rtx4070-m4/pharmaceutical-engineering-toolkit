📖 Overview

Pharmacognosy & Phytochemistry Toolkit is a standalone scientific web application designed to perform essential laboratory calculations used in herbal drug evaluation and phytochemical analysis.

Pharmacognosy laboratories routinely perform repetitive calculations when analyzing crude plant materials, extracts, and natural products. This toolkit simplifies those tasks by providing a centralized interface for common pharmacognostic calculations.

The application is built as a single-file responsive HTML application that runs entirely in the browser with no external libraries, frameworks, or installations required.

This makes it ideal for:

🧪 Pharmacognosy laboratories
🌿 Herbal drug research
💊 Pharmacy education
🔬 Phytochemistry studies
📊 Natural product analysis

🧮 Calculators Included

The toolkit currently supports the following pharmacognosy calculations:

Calculator	Description
Extract Yield	Calculates percentage yield of plant extract
Moisture Content	Determines water content in crude drugs
Ash Value	Measures inorganic residue in plant material
Swelling Index	Evaluates mucilage-containing drugs
Hemolytic Index	Determines hemolytic activity of saponins
Bitterness Value	Estimates bitterness relative to quinine
Extractive Value	Measures soluble constituents in solvents
🔬 Scientific Formulas Implemented
Extract Yield
Extract Yield (%) = (Weight of Extract / Weight of Crude Drug) × 100
Moisture Content
Moisture Content (%) =
((Initial Weight − Final Weight) / Initial Weight) × 100
Total Ash
Total Ash (%) = (Weight of Ash / Weight of Sample) × 100
Extractive Value
Extractive Value (%) =
(Weight of Extract / Weight of Air-Dried Drug) × 100
